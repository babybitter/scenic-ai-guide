# 登录页景区背景图

把景区实景图放在这个目录，命名为 **`login-bg.png`**：

```
web/public/images/login-bg.png
```

登录 / 注册 / 忘记密码页左侧会自动使用它（`LoginLeftView.vue`）。

## 尺寸建议

登录页左侧面板约为屏幕的 `65vw × 100vh`（≤1600px 宽时为 `60vw`，≤1180px 窄屏隐藏），
在 1920×1080、2560×1440 等常见屏幕上比例约 **1.15:1（接近正方形、略宽）**，所以不要用严格的 1:1。

- 推荐：**横向实景图，`cover` 填充**，宽度至少 **1600px**，最好 **2000–2560 × 1600（约 4:3 / 3:2）**。
- 主体（如灵山大佛）尽量居中/偏上，因为底部约 80px 会叠加标题与副标题文字。
- 文件未就位时不会报错，会回退到主题墨绿底色 + 暗色蒙层。

如果你更想用 `.png` / `.webp`，或想改成从 `src/assets/images/login/` 走 Vite 哈希打包，
告诉我，我改一行引用即可。
