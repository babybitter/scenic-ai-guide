# Guide chat background

Place the optional chat artwork here as `chat-bg.webp`.

- Recommended source size: **1600 × 1600 px** (1:1).
- Keep important details inside the centered 60% safe area because the chat panel is responsive.
- Prefer a low-contrast image without text and export WebP at 300–500 KB or less.
- Recommended rendered opacity: 4–7% in light mode and 3–5% in dark mode.
- Use `background-size: cover` and a neutral color overlay so message bubbles remain the visual focus.

The chat viewport has no fixed pixel size. At common desktop resolutions it is usually about 640–900 px wide and 550–760 px high, while mobile uses the available screen width.
